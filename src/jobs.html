mod_perl @ Geizhals.at

23.3.2006

Die Preisvergleich Internet Services AG (Geizhals.at) zählt zu den größten Preisvergleichsportalen im deutschsprachigen Raum. Als zukunftssicherer Dienstgeber in einer enorm interessanten, dynamischen Branche bieten wir engagierten Personen vielfältige Berufs- und Karrieremöglichkeiten.

Weiters zeichnen uns Sozialleistungen, Weiterbildungsmöglichkeiten, ein junges Team, ein sehr zentraler Arbeitsplatz (1020 Wien) sowie ein angemessenes Entgelt aus.

Ihre Aufgaben:

Zu Ihren Aufgaben wird vor allem die Entwicklung und Wartung des Website-Backendbereiches, sowie die Neuentwicklung interner Software gehören.

Ihr Profil:

  • mindestens 5 Jahre Entwicklungserfahrung mit Perl
  • Web Applikationen mit Apache / mod_perl, shell Scripts
  • use DBI; use LWP; use CGI;
  • gute Kenntnisse von regular expressions und references
  • zumindest Basiskenntnisse in OO Perl
  • Erfahrung beim Einlesen in bestehenden Code, sowie Wartung von bestehendem Code
  • Datenbankerfahrung in SQL. Verwendet wird PostgreSQL, daher sollten Begriffe wie Constraints, Stored Procedures oder Trigger keine Fremdwörter für Sie sein.
  • Linux Erfahrung, Anwendung gängiger UNIX tools (z.B. ssh, rsync, RCS...)
  • Grundkenntnisse bez. Security in Web-Anwendungen und sonstigen Netzwerkapplikationen (zb. SQL injection)
  • Teamfähigkeit

Wenn Sie diese Anforderungen erfüllen und zudem gewohnt sind, sehr eingeständig zu arbeiten, sowie eine hohe Eigenmotivation an den Tag legen, bewerben Sie sich bei uns!

Bitte richten Sie Ihre Bewerbung inklusive Lebenslauf, am besten gleich mit aussagekräftigen Codebeispielen, ausschliesslich per Mail an: jobs@geizhals.at
